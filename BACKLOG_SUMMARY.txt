================================================================================
KEY FEATURES
================================================================================

✓ Manage part maintenance templates (CRUD + archive/restore)
✓ Register and manage multiple cars
✓ Customize maintenance intervals per car
✓ Track maintenance service changes (km or date-based)
✓ View maintenance history for each part
✓ Dashboard with color-coded status (overdue/due soon/OK)
✓ Quick mileage updates from dashboard

================================================================================
VALIDATION RULES
================================================================================

Frontend:
- Date format: dd/mm/yyyy (dd ≤ 31, mm ≤ 12, yyyy ≥ 2000)
- Km values: >= 0
- Interval values: > 0
- Interval type: dropdown (km / time only)

Backend:
- Parse dd/mm/yyyy to datetime
- Calculate next service based on interval type
- Sort all lists by createdAt DESC (newest first)

Status Calculation:
- Km-based: Red (≥nextKm), Yellow (<300km), Green (≥300km)
- Time-based: Red (≥nextDate), Yellow (<30 days), Green (≥30 days)
